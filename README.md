# Códice Oscuro

Grimorio ritual BDSM para gestión de cultos, jerarquías y dinámicas D/s consensuadas.

## 🕯️ Características

- **Gestión de Cultos**: Crea y administra tu propio culto con estructura jerárquica
- **Roles Duales**: Deidad (dominante/administrador) y Fiel (sumiso/miembro)
- **Sistema de Invitaciones**: Códigos permanentes para admitir nuevos miembros
- **Jerarquía Visual**: Árbol invertido mostrando relaciones de subordinación
- **Tareas y Recompensas**: Sistema de puntos de fe, premios y consecuencias
- **Sesiones BDSM**: Control de beats, tarjetas de acciones y mensajes de audio
- **Estética Gótica**: Diseño de grimorio oscuro con animaciones rituales

## 🚀 Stack Tecnológico

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Backend**: Supabase (Auth, Database, Storage)
- **Animaciones**: Framer Motion

## 📖 Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
